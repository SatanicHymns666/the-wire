# The Feed

The Feed is a daily automated sweep of the world's wreckage. No editorializing. No advocacy. No predictions. Just the record — collected every day, organized by category, filed and forgotten until something rhymes with it later.

The witness role belongs to the scribes of the destruction gods — Kali devouring time, Apophis circling the sun, Tiamat split open, Set scattering the pieces. They watched the world tear itself apart and wrote it down. This is that, minus the theology. The deities are not in these pages. Only what they would recognize: the bridge that gave way, the fire that wouldn't stop, the ship that sank under circumstances nobody is explaining.

Every day the net goes out across thirteen categories: Earth, Water, Sky, Infrastructure, Aviation, Maritime, Violence, Public Spaces, Political, Economic, Health, Space, and Unexplained. What comes back is recorded with headline, keywords, source, and link. Nothing is summarized. Nothing is interpreted. You read the bones yourself and decide what animal they came from.

This is an automated system. It runs daily. Each file is a dated snapshot stored in the `feed/` directory and never deleted. The scribe does not editorialize. The archive does not forget.
